# PWA 설치 가이드 (스마트폰)

## 📱 스마트폰에서 PWA 설치 방법

### Android (Chrome/Edge)

1. **브라우저에서 접속**
   - Chrome 또는 Edge 브라우저로 `https://travel23.mycafe24.com/manager-app/` 접속

2. **설치 방법 (3가지)**
   
   **방법 1: 주소창 설치 아이콘**
   - 주소창 오른쪽에 나타나는 "설치" 또는 "추가" 아이콘 클릭
   - "홈 화면에 추가" 또는 "설치" 선택

   **방법 2: 메뉴에서 설치**
   - 브라우저 메뉴(⋮) 클릭
   - "앱 설치" 또는 "홈 화면에 추가" 선택

   **방법 3: 설치 페이지 사용**
   - `https://travel23.mycafe24.com/manager-app/install` 접속
   - "앱 설치하기" 버튼 클릭 (지원되는 경우)

3. **설치 확인**
   - 홈 화면에 "Hangbok77" 아이콘 생성
   - 아이콘을 탭하면 앱처럼 실행됨

### iOS (Safari)

1. **Safari 브라우저에서 접속**
   - `https://travel23.mycafe24.com/manager-app/` 접속

2. **홈 화면에 추가**
   - 하단 공유 버튼(□↑) 클릭
   - "홈 화면에 추가" 선택
   - 이름 확인 후 "추가" 클릭

3. **설치 확인**
   - 홈 화면에 "Hangbok77" 아이콘 생성
   - 아이콘을 탭하면 앱처럼 실행됨

## ⚠️ 중요 사항

### PWA 설치 조건

1. **HTTPS 필수**
   - 카페24 호스팅에서 HTTPS를 사용해야 합니다
   - HTTP에서는 PWA 설치가 제한될 수 있습니다

2. **필수 파일**
   - `manifest.webmanifest` ✅ (있음)
   - `sw.js` (Service Worker) ✅ (있음)
   - PWA 아이콘 이미지 ⚠️ (없음 - 생성 필요)

3. **PWA 아이콘 필요**
   - `pwa-192x192.png` (192x192 픽셀)
   - `pwa-512x512.png` (512x512 픽셀)
   - 이 파일들이 없으면 설치 시 기본 아이콘이 표시됩니다

## 🔧 현재 상태

✅ **설정 완료:**
- manifest.webmanifest 파일 있음
- Service Worker 설정됨
- PWA 설치 페이지 있음

⚠️ **추가 필요:**
- PWA 아이콘 이미지 파일 (pwa-192x192.png, pwa-512x512.png)

## 📝 PWA 아이콘 생성 방법

1. **아이콘 이미지 준비**
   - 192x192 픽셀 이미지
   - 512x512 픽셀 이미지
   - PNG 형식, 투명 배경 가능

2. **파일 위치**
   - `manager-app/public/pwa-192x192.png`
   - `manager-app/public/pwa-512x512.png`
   - 또는 `manager-app/dist/` 폴더에 직접 추가

3. **빌드 후 확인**
   - 빌드 후 `dist/` 폴더에 아이콘 파일이 있는지 확인
   - 호스팅 서버의 `/manager-app/` 폴더에도 업로드

## 🚀 설치 테스트

1. 호스팅 서버에 모든 파일 업로드 완료
2. 스마트폰 브라우저에서 접속
3. 설치 옵션이 나타나는지 확인
4. 설치 후 홈 화면에서 실행 테스트
